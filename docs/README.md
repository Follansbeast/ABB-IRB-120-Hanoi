## Tower of Hanoi Specifications
- Spacing Rod to Rod (~50-55mm center)
- A to B: ~2 ⅜”
- B to C: ~3 7/16”
- 10 rings
- Smallest Ring Diameter (19.05mm, 0.750”)
- Largest Ring Diameter (47.625mm, 1.875”)
- Post Diameter (6.35mm, 0.25")
- Ring Height (6.35mm, 0.25")
